🧠 MemoVault – Personal Digital Diary Application

MemoVault is a desktop-based Personal Digital Diary Application developed as a Semester-End Project for the Object-Oriented Programming (OOP) course.
It is built using C++ and Qt Creator, combining strong OOP principles with a modern graphical user interface.

The application provides a secure and user-friendly platform to create, manage, and store personal diary entries digitally.

🎯 Project Objectives
Apply Object-Oriented Programming concepts in a real-world project
Gain hands-on experience with Qt Creator & Qt Framework
Develop an interactive GUI-based desktop application using C++
Design a user-friendly system for managing personal diary entries
