## 📖 Overview

**MemoVault** is a desktop-based Personal Digital Diary Application developed as a **Semester-End Project** for the **Object-Oriented Programming (OOP)** course. Built using **C++** and **Qt Creator**, it combines strong OOP principles with a clean, modern graphical interface — giving users a secure and intuitive platform to create, manage, and store personal diary entries.

---

## 🎯 Project Objectives

- Apply **Object-Oriented Programming** concepts in a real-world project
- Gain hands-on experience with **Qt Creator & Qt Framework**
- Develop an interactive **GUI-based desktop application** using C++
- Design a user-friendly system for managing personal diary entries

## 🚀 Getting Started

### 📌 Prerequisites

- [Qt Creator](https://www.qt.io/product/development-tools)
- Qt Framework (5.x or later recommended)
- C++ Compiler — **MinGW** or **MSVC**

### ⚙️ Installation & Setup

```bash
# 1. Clone the repository
git clone https://github.com/your-username/MemoVault.git

# 2. Open Qt Creator
# 3. Click File → Open Project
# 4. Navigate to the cloned folder and select the .pro file
# 5. Choose your Qt Kit (MinGW / MSVC)
# 6. Click Build → Run (Ctrl + R)
```
