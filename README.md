📝 MemoVault – Personal Digital Diary Application

MemoVault is a desktop-based Personal Digital Diary Application developed as a Semester-End Project for the Object-Oriented Programming (OOP) course. The project was built using C++ and Qt Creator, combining object-oriented principles with an interactive graphical user interface.

The application provides a secure and organized platform where users can record, manage, and preserve their personal thoughts and experiences digitally.

🎯 Project Objectives
Apply Object-Oriented Programming concepts in a real-world application.
Gain hands-on experience with Qt Creator and Qt Framework.
Develop an interactive GUI-based desktop application using C++.
Design a user-friendly system for managing personal diary entries.

🚀 Getting Started
Prerequisites

Ensure the following are installed on your system:

Qt Creator
Qt Framework
A compatible C++ compiler (MinGW/MSVC)
⚙️ Installation and Setup
1. Clone the Repository
git clone https://github.com/your-username/MemoVault.git
2. Open the Project
Launch Qt Creator
Select Open Project
Navigate to the MemoVault project folder
Open the .pro file
3. Configure the Project
Choose an appropriate Qt Kit
Build the project
4. Run the Application
Click the Run button in Qt Creator to launch MemoVault.
